# How to run this PWA

In the project directory, you can run:

### `npm install`

... then

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

# Features and Frameworks used:

- Create React App
- React
- React Scripts
- React DOM
- Pigeon Maps
- React Suite
- Font Awesome Icons
